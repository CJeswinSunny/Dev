# My Portfolio Website

A clean, modern portfolio website built with HTML and CSS.

## Files

- **index.html** - Main portfolio page
- **style.css** - Styling and responsive design

## Features

- Responsive design (mobile, tablet, desktop)
- Modern gradient color scheme
- Smooth animations and hover effects
- Navigation bar with smooth scrolling
- About, Projects, and Contact sections
- Professional footer with social links

## Sections

1. **Hero Section** - Eye-catching introduction
2. **About Me** - Personal introduction and skills
3. **Projects** - Showcase of 3 project examples
4. **Contact** - Contact form for inquiries
5. **Footer** - Social links and copyright

## How to Use

1. Open `index.html` in your web browser
2. Customize the content with your own information:
   - Change "Your Name" to your actual name
   - Update project descriptions and links
   - Add your social media links
   - Modify colors in `style.css` if desired

## Customization

Edit the CSS variables in `style.css` (lines 8-13) to change the color scheme:
- `--primary-color` - Main accent color
- `--secondary-color` - Gradient second color
- `--text-color` - Text color
- `--light-bg` - Light background color

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
